\# Assets



This folder contains supporting files used across multiple labs in this repository.



These assets are separated from individual labs to keep lab folders clean and to centralize reusable resources.



\## Folder Breakdown



\- configs  

&nbsp; Contains saved device configurations used for reference, backup, or reuse across labs.



\- diagrams  

&nbsp; Stores network diagrams and topology visuals created during lab planning and implementation.



\- resume  

&nbsp; Includes resume versions and related portfolio assets used for job applications.



\- screenshots  

&nbsp; Contains general screenshots that are not tied to a specific lab’s step-by-step evidence.



\## Purpose



The goal of this folder is to organize reusable and supporting materials in one place, separate from individual lab documentation, while maintaining a clean and scalable repository structure.
