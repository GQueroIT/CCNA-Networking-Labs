\# Templates



This folder contains reusable templates used to maintain consistency across all labs in this repository.



These templates help ensure that each lab follows the same structure, making the work easier to read, review, and scale over time.



\## Included Templates



\- LAB-TEMPLATE.md — Standard structure for lab documentation

\- TROUBLESHOOTING-TEMPLATE.md — Used for documenting issues and resolution steps

\- CHANGELOG-TEMPLATE.md — Tracks updates and improvements to labs

\- README-STYLE.md — Maintains consistent formatting and writing style across the repository



\## Purpose



The goal of these templates is to keep documentation clean, repeatable, and professional across all networking labs.
