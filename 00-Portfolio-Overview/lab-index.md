## Objective

This portfolio represents my hands-on progression through core networking concepts, from foundational device configuration to advanced routing and network design.

Each lab is structured to simulate real-world scenarios, focusing on implementation, verification, and troubleshooting rather than theory alone. The goal of this repository is to demonstrate practical networking skills aligned with real IT infrastructure environments.

---

## What This Portfolio Demonstrates

- Practical experience configuring Cisco networking devices using CLI
- Strong understanding of VLAN segmentation, trunking, and Layer 2 design
- Implementation of routing protocols including OSPF (single-area and multi-area)
- Ability to design scalable network topologies
- Troubleshooting connectivity, VLAN, and routing issues
- Structured documentation and real-world lab workflow

---

## Learning Progression

This lab series is designed to build networking knowledge in layers:

- **Fundamentals** -> Core concepts (OSI model, subnetting, IPv6)
- **Switching** -> VLANs, trunking, and Layer 2 behavior
- **Routing** -> Static routing, inter-VLAN routing, OSPF, and multi-area design
- **Services & Security (Upcoming)** -> Network services and hardening
- **Troubleshooting** -> Real-world issue resolution scenarios

Each section builds on the previous one, reinforcing both configuration skills and design understanding.
